自定义构造型与领域特定建模扩展
定义 agent-wrapper 项目中的 UML 扩展：Provider 构造型、Agent Interface 约束、Event 类型枚举。适用于领域特定建模。
CUSTOM STEREOTYPES

构造型定义

  • «provider» — 标记 Agent CLI 适配实现类，带 protocol 标签
  • «agentInterface» — 标记可流式/支持工具调用的统一接口
  • «event» — 标记事件数据类型，带 sessionScoped 标签
  • «eventType» — 标记事件类型枚举（5 种值）
  • «handler» — 标记生命周期回调类型（pre/post/inline）
PROFILE APPLICATION

应用示例

  • ClaudeCodeAgent 应用 «provider», protocol="stream-json"
  • CodexAgent 应用 «provider», protocol="json"
  • Event 应用 «event», sessionScoped=true
  • ApprovalHandler 应用 «handler», lifecycle="inline"
