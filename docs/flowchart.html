Agent Wrapper 流程图

Orchestrator.Run() 主执行流程
展示 Orchestrator.Run() 的完整执行路径：输入校验 → Agent 启动 → 事件循环 → 审批/预算/重试 → 终止。
PRIMARY PATH

主流程

  • 输入校验 → runAgentWithRetry → 事件循环
  • text_delta → 直接转发到输出 channel
  • tool_call → 审批决策 → allow/deny/abort
  • turn_end → 预算检查 → 继续或终止
EXCEPTION PATHS

异常分支

  • Prompt 为空 → 立即返回 error
  • Tool denied → 注入 DENIED tool_result
  • Tool abort → 设置 stop_reason="aborted"
  • Context exceeded → 压缩 + 重试（最多3次）
  • Budget exceeded → 终止运行
LOOP CONTROL

循环终止条件

  • stop_reason = "end_turn" / "stop"
  • turn_number >= MaxTurns
  • approval = abort
  • context.Done() 取消
