Agent Wrapper 系统架构

统一 Coding Agent 包装库 — 8 个 Provider，1 个接口
Go 语言库。通过统一 Agent 接口驱动 Claude Code、Cursor、Kimi Code、Antigravity、Codex、Pi、OpenCode、ACP 等 CLI，提供多 turn 编排、审批、预算控制与上下文压缩重试。

核心组件

  • Agent — 统一接口，8 个 Provider 实现
  • Orchestrator — 多 turn 编排 + 审批 + 预算
  • Registry — Factory 注册表，按名创建 Agent

支撑模块

  • Harness — Approval/Budget/ContextCompressor
  • Process — 子进程管理 + SIGTERM/KILL
  • Types — Event/Message/TokenUsage/Errors

Provider 协议

  • stream-json (NDJSON): Claude/Cursor/Kimi
  • JSON-RPC: ACP
  • JSON: Codex/Pi/OpenCode
  • Plain text: Antigravity
