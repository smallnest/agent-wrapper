Agent Wrapper 活动图

单次 Agent 调用的完整活动流程与阶段划分
按阶段展示 Orchestrator 如何处理一次 Agent 调用：校验 → 启动 → 事件处理 → 审批/预算 → 终止。
ACTIVITY PHASES

活动阶段

  • Phase 1: 输入校验 — 检查 Prompt 非空
  • Phase 2: Agent 启动 — 构建参数、启子进程、初始化扫描器
  • Phase 3: 事件循环 — 按类型分派，审批/预算介入
  • Phase 4: 终止清理 — SIGTERM → 回收进程
PARALLEL ACTIVITIES

并行活动

  • 子进程启动 + 扫描器初始化 + 上下文监听（3路并行）
  • goroutine 分开读取 stdout frames 和执行事件循环
  • ctx cancellation → 独立 goroutine 发送 SIGTERM
