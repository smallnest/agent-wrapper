Agent Wrapper 泳道图

跨组件职责流程：从调用到完成的完整路径
按 Caller / Orchestrator / Agent / Process / CLI 5 个泳道展示一次完整调用的所有步骤及跨泳道交互。
SWIMLANES

角色与职责

  • Caller — 发起调用，消费事件流
  • Orchestrator — 校验、审批调度、预算控制
  • Agent — 参数构建、子进程管理、协议解析
  • AgentProcess — 子进程生命周期
  • Agent CLI — LLM 交互，产 NDJSON 输出
CROSS-LANE INTERACTIONS

跨泳道交互

  • 1→2: Caller → Orchestrator (调用入口)
  • 2→3→4→5: 下行链 (启动层级)
  • 5→6→7→8→9: 上行链 (事件传播)
  • 10: 清理信号 (SIGTERM 广播)
