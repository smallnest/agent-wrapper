AgentProcess 子进程生命周期状态变迁
展示 AgentProcess 包装的子进程从创建到销毁的全部状态及触发事件。也覆盖 Orchestrator turn loop 的关键状态。
AGENTPROCESS STATES

子进程生命周期

  • Created → cmd.Start() 创建进程
  • Running → stdin/stdout 管道活跃
  • Exited OK → 正常退出 (exit 0)
  • Exit Error → 非零退出码 + stderr
  • Terminated → SIGTERM → (5s) → SIGKILL
ORCHESTRATOR STATES

Turn 生命周期

  • Idle → Run() → Processing
  • Processing → turn_end → Completed (normal)
  • Processing → more turns → Processing (loop)
  • Processing → error/abort → Aborted
  • Processing → ctx exceeded → compress+retry (self)
